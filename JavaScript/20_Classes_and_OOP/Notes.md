# OOP

## Parts of OOP

constructor function
prototype
classes
instances(new, this)

## 4 Pillars

Abstraction: Hiding the internal implementation details and exposing only the essential functionality to the user, like in fetch we are
not given the detailed info about whats happening under the hood

Encapsulation: Encapsulation is the process of bundling data (properties) and the methods that operate on that data into a single unit (class), while restricting direct access to the internal data. (Its a method of protecting the data)

Inheritance: Inheritance allows a child class (derived class) to reuse the properties and methods of a parent class (base class).

Polymorphism: Polymorphism allows different classes to use the same method name while providing their own specific implementation.
